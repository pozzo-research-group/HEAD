# Genetic_Algorithm

This repository contains code to perform batch-wise optimization where a spectra or scattering curve is targeted. A genetic algorithim is used
to for the optimization and emulators are used for  virtual experiments to test the performance of the GA. The emulators consist of pyGDM2
which models the extinction spectra of metallic nanostructures and Sasmodels which is used to caluclate the theoretical scattering curve
of a predetermined structure. In the branches of this repository there is additional code that uses Bayesian Optimization for the same
function.
